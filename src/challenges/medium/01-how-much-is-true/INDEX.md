# How Much Is True?

Create a function which returns the number of `true` values there are in an array.

---

### Examples:

```
howMuchIsTrue([true, false, false, true, false]) ➞ 2
howMuchIsTrue([false, false, false, false]) ➞ 0
howMuchIsTrue([]) ➞ 0
```
